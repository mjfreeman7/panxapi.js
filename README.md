# panxapi.js

panxapi.js is a simple [Node.js](http://nodejs.org/) client for Palo Alto Networks firewall API.

## Dependencies

 - PAN-OS 4.0 or greater
 - Node.js

## Installation

    npm install panxapi

## Further reading

Reviewing Palo Alto's documentation on their API is recommended.
